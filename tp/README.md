# TP Python IA20

Ce dossier contient les 6 TPs Python du module IA20. L'idée est de rester très progressif, avec peu de maths et beaucoup de manipulation concrète.

- `TP1_intro_donnees` : prise en main Python/notebooks & exploration de données.
- `TP2_regression_lineaire` : régression linéaire avec scikit-learn.
- `TP3_classification_knn` : classification KNN et lien avec la démo Observable.
- `TP4_clustering_kmeans` : clustering K-Means et visualisation.
- `TP5_mlp` : petit réseau de neurones (MLP) sur problème jouet.
- `TP6_mini_projet` : mini-projet encadré, pipeline IA de bout en bout.

Les notebooks pourront être créés au format `.ipynb` dans chaque sous-dossier (par ex. `TP1_intro_donnees/TP1.ipynb`).
